# LLM 仿真调研仓库

本仓库收录了一份对十三个大型语言模型（LLM）训练与推理仿真/优化项目的对比调查表 `survey.csv`，帮助研究者快速了解相关工具的功能特性、使用条件与适用场景。

## 数据内容概览

`survey.csv` 以列为维度、行表示项目，汇总以下关键信息：

| 字段 | 说明 |
| --- | --- |
| 对比维度 | 便于在表格顶部快速定位项目名称。 |
| 类别 | 标识项目面向训练还是推理。 |
| 机构/团队 | 研发团队或合作单位。 |
| 论文来源 | 对应的论文或公开材料，方便进一步查阅。 |
| 开源仓库 | 工具是否开源、代码获取方式及成熟度。 |
| 输入数据需求 | 建模或仿真所需的日志、测量数据或配置文件等。 |
| 输出指标 | 能够给出的性能或成本评估指标，例如吞吐、延迟、能耗。 |
| 建模方法 | 使用的仿真/建模手段（解析模型、事件驱动仿真等）。 |
| 建模粒度 | 关注的层级（算子、阶段、集群等）。 |
| 硬件依赖 | 运行仿真时依赖的硬件或系统环境。 |
| 校准与测量方式 | 精度校准或数据采集方式。 |
| 特色与寻优机制 | 各项目的亮点、优化策略与自动调参能力。 |
| 精度与缺陷 | 已报告的准确度表现或局限性。 |

> 提示：CSV 使用 UTF-8 编码并包含列标题，可直接用 Excel、LibreOffice Calc 或 `pandas` 读取。

## 使用建议

- **快速调研**：筛选满足自身需求的仿真或优化框架，初步了解其覆盖能力。
- **需求评估**：在复现、迁移或对接前，确认所需输入数据与依赖。
- **方案对比**：依据建模粒度与精度，权衡不同工具在研发、实验和生产场景中的适配性。

若需程序化处理数据，可参考以下 Python 示例：

```python
import pandas as pd

df = pd.read_csv("survey.csv")
print(df.columns)
```
